Komplex

TECHNICAL_SOLUTIONS

✦ SIMD Parallelization

AVX2 — four complex number iterations simultaneously per core. 4× throughput vs scalar.

✦ Multi-threading

Dynamically partitions screen into tiles, distributing across all logical processors.

✦ Precision

Custom fixed-point math — zoom stability beyond standard double precision limits.

function applyLang(lang) {
  document.querySelectorAll('[data-en]')
    .forEach(el => {
      el.textContent = lang === 'cz'
        ? el.dataset.cz
        : el.dataset.en;
    });
}
